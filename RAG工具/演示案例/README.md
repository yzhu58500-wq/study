# Word 图片批量插入工具

一个 Python 脚本，用于自动将指定文件夹下的图片批量插入到 Word 文档的表格中。

## 功能特点

- 🚀 自动遍历图片文件夹，按编号顺序处理
- 📍 自动识别表格目标行的最左和最右单元格
- 📐 图片等比例缩放，高度可调
- 📄 支持现有 Word 文档追加模式
- 🔧 配置简单，无需修改代码

## 安装依赖

```bash
pip install python-docx pillow
```

## 快速开始

### 1. 准备工作

**图片文件夹结构：**
```
图片文件夹/
├── 1/
│   ├── xxx.jpg
│   └── yyy.png
├── 2/
│   ├── aaa.jpg
│   └── bbb.png
├── 3/
│   └── ...
└── ...
```

- 文件夹名称以数字开头（如 `1`、`2`、`3` 或 `01_xxx`、`02_xxx`）
- 每个文件夹内包含 **2 张图片**（支持 `.jpg`、`.jpeg`、`.png`）
- 图片按文件名排序，第 1 张插左边，第 2 张插右边

**Word 文档要求：**
- 文档中已有若干表格
- 每个表格的目标行有两个独立单元格（或更多）

### 2. 修改配置

打开 `word图片批量插入.py`，修改顶部 `CONFIG`：

```python
CONFIG = {
    # 路径配置
    "source_folder": "./你的图片文件夹",   # 图片文件夹路径
    "template_file": "./你的文档.docx",    # Word 文档路径
    "output_file": "./输出文档.docx",      # 输出文件路径
    
    # 图片配置
    "image_height": 5.0,                   # 图片高度（cm）
    
    # 表格定位配置
    "target_row_position": -2,             # 目标行：-2 表示倒数第二行
}
```

### 3. 运行脚本

```bash
python word图片批量插入.py
```

## 配置说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `source_folder` | 图片文件夹路径（存放所有子文件夹的根目录） | `./图片文件夹` |
| `template_file` | 现有 Word 文档路径 | `./模板文档.docx` |
| `output_file` | 输出 Word 文件路径 | `./输出文档.docx` |
| `image_height` | 图片固定高度（厘米），宽度自动等比缩放 | `5.0` |
| `target_row_position` | 目标行位置，负数表示倒数 | `-2` |

### 行位置说明

| 值 | 含义 |
|---|---|
| `-1` | 倒数第 1 行（最后一行） |
| `-2` | 倒数第 2 行 |
| `-3` | 倒数第 3 行 |
| `0` | 第 1 行 |
| `4` | 第 5 行 |

**推荐使用负数**，这样表格行数变化时无需修改配置。

## 工作原理

```
┌─────────────────────────────────────────┐
│            Word 文档（多个表格）           │
├─────────────────────────────────────────┤
│  表格 1                                  │
│  ┌──────────┬──────────┬──────────┐     │
│  │   ...    │   ...    │   ...    │     │
│  ├──────────┼──────────┼──────────┤     │
│  │  左图←   │   ...    │  →右图   │ ← 目标行
│  └──────────┴──────────┴──────────┘     │
├─────────────────────────────────────────┤
│  表格 2                                  │
│  ┌──────────┬──────────┐                │
│  │  左图←   │  →右图   │ ← 目标行        │
│  └──────────┴──────────┘                │
└─────────────────────────────────────────┘
```

1. 脚本打开现有 Word 文档
2. 遍历所有表格，定位到每个表格的目标行
3. 自动识别该行的**最左单元格**和**最右单元格**
4. 将图片文件夹 1 的两张图插入表格 1
5. 将图片文件夹 2 的两张图插入表格 2
6. 以此类推...

## 注意事项

1. **图片数量**：每个文件夹必须有 2 张图片，否则会跳过
2. **表格数量**：图片文件夹数量不应超过表格数量
3. **单元格结构**：目标行至少要有 2 个独立单元格
   - 如果整行是合并单元格，两张图会叠在一起
4. **文件格式**：支持 `.jpg`、`.jpeg`、`.png` 格式

## 常见问题

### Q: 提示 "No module named 'docx'"

```bash
pip install python-docx
```

### Q: 只插入了左图，右图没插入？

检查：
1. 文件夹里是否有 2 张图片
2. 目标行是否有 2 个独立单元格（不是合并单元格）

### Q: 图片太大或太小？

调整配置中的 `image_height` 值（单位：厘米）。

### Q: 插入到了错误的行？

修改 `target_row_position`：
- 用负数表示倒数第几行
- 或用正数表示具体第几行（从 0 开始）
