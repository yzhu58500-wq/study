## 简介

这个仓库沉淀和总结我在 AI 应用开发过程中积累的一些模块化代码和辅助工具。主要聚焦于两个方向：

- **RAG 系统构建**：文档处理、向量检索、知识库管理等相关模块
- **Agent 编排**：多 Agent 协同、任务调度、工具调用等辅助工具

初衷是方便自己下次使用时快速复用，同时也作为学习过程的记录。代码会在实践中持续总结和更新。
